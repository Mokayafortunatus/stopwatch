1. **Clone the repository** (if you haven't already):
   ```bash
   git clone <repository-url>
   cd stopwatch
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the project**:
   ```bash
   npm start
   ```

   ```bash
   pm2 start index.js --name stopwatch
   ```
4. **Configuration**
